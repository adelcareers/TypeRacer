# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-01-12
### Added
- Responsive, accessible UI shell with structured layout and typography.
- Typing test engine with start flow, timer on first input, and completion rules.
- Difficulty selection with multiple text pools and passage switching.
- Real-time per-character feedback (correct/incorrect/current).
- Results calculation for WPM, accuracy, and elapsed time.
- Best WPM persistence per difficulty via localStorage.
- Instructions modal with accessible open/close behavior.
- Retry flow that refreshes the passage.
